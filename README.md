# Timer_ADC_STM-32
STM32 Black Pill project implementing a 100 Hz timer-triggered ADC system. ADC conversions are started on timer overflow, results are transferred using DMA, and sampled data is sent via UART and displayed on Windows Terminal, demonstrating efficient TIM-ADC-DMA-UART integration.
